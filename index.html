What is a Hash Table?

A hash table maps keys to indices using a hash function. This app uses a polynomial rolling hash and resolves collisions with separate chaining. When the load factor grows beyond a threshold, the table resizes to keep operations fast.

  • Key: Student ID
  • Value: Student (Name, Age, Grade, Major)
  • Collision handling: Separate chaining
  • Resize policy: Automatic when load factor exceeds 0.75
